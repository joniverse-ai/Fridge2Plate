# 냉장고를털어라 (ClearFridge) — PRD (Product Requirements Document)

## 한 줄 요약

냉장고 속 재료를 입력하면 만들 수 있는 한국 가정식 레시피를 찾아주는 웹앱.

## 문제 정의

> [우리 가족]이 [식사때가 되어서 냉장고를 열었을 때] [재료의 종류는 많지만 레시피를 몰라서] [뭘 해서 먹어야 할지 모르는 불편]을 겪는다.

## 사용자

- **주 사용자**: 가족 구성원 (모바일 위주 사용)
- **사용 맥락**: 식사 준비 직전, 냉장고 앞에서

## 핵심 흐름 (Core Flow)

```
냉장고 열기 → 재료 입력 → 레시피 검색 → 부족 재료 확인 → 요리 시작
```

### 상세 플로우

1. **재료 입력**: 텍스트로 재료명을 입력하고 태그로 추가
2. **레시피 검색**: 입력된 재료 기반으로 매칭되는 레시피 목록 표시
3. **레시피 상세**: 선택한 레시피의 재료, 조리법, 이미지 확인. 동일 레시피의 유튜브 영상이 있으면 함께 제공
4. **부족 재료 체크**: 메인 재료는 사용자가 입력한 재료의 비중을 크게 가져가고, 부족 재료 확인은 소스·양념류를 위주로 표시하여 확인받는다

## 기능 요구사항

### Must Have (MVP)

| # | 기능 | 설명 |
|---|------|------|
| M1 | 재료 태그 입력 | 재료명을 입력하고 태그로 추가/삭제 |
| M2 | 레시피 매칭 검색 | 입력된 재료 기준으로 만들 수 있는 레시피 목록 |
| M3 | 매칭률 표시 | 내 재료로 몇 %를 충족하는지 표시 |
| M4 | 레시피 상세 보기 | 재료, 조리 순서, 이미지, 유튜브 영상 링크 |
| M5 | 부족 재료 확인 | 소스·양념류 위주로 부족 재료를 표시하고 "있어요" 체크 가능 |
| M6 | 반응형 디자인 | PC와 모바일 모두 사용 가능한 반응형 레이아웃 |

### Nice to Have (v2)

| # | 기능 | 설명 |
|---|------|------|
| N1 | 냉장고 저장 | 내 냉장고 재료를 저장해두고 다음에 불러오기 |
| N2 | 즐겨찾기 | 마음에 드는 레시피 저장 |
| N3 | 재료 자동완성 | 입력 시 재료명 자동 추천 |
| N4 | 카테고리 필터 | 한식/중식/양식, 조리시간, 난이도 |

## 데이터 소스

- **식약처 COOKRCP01**: 공공데이터포털(data.go.kr) 조리식품 레시피 DB
  - 레시피명, 조리방법, 재료, 조리순서(20단계), 이미지, 열량/영양정보
  - 약 1,000여 개 한식 레시피
- **YouTube Data API**: 레시피명으로 유튜브 검색 → 관련 영상 링크 제공
- **데이터 전략**: 식약처 API에서 레시피를 Supabase DB로 가져와 저장 후 자체 검색, 유튜브 영상은 레시피명으로 실시간 검색

## 화면 구성

> 모든 화면은 PC(데스크톱)와 모바일 모두에서 사용 가능한 **반응형**으로 제작

### 1. 메인 페이지 (`/`)
- **랜딩 화면**: 세련된 주방에 놓인 냉장고 이미지로 시작
- **인터랙션**: 냉장고를 클릭하면 열리는 애니메이션과 함께 재료 입력 UI가 등장
- 재료 입력 영역 (검색창 + 태그)
- "레시피 찾기" 버튼

### 2. 검색 결과 (`/recipes`)
- 매칭률 순으로 정렬된 레시피 카드 목록
- 각 카드: 이미지, 레시피명, 매칭률, 조리방법 태그

### 3. 레시피 상세 (`/recipes/[id]`)
- 레시피 이미지
- 유튜브 영상 (동일 레시피가 유튜브에 있으면 임베드)
- 재료 목록 (보유/미보유 구분)
  - 메인 재료: 사용자 입력 재료 우선 매칭
  - 소스·양념류: 부족 시 "있어요?" 체크 UI
- 조리 순서 (단계별 이미지 포함)
- 영양 정보

## 성공 지표

- 가족이 실제로 이 앱을 써서 저녁 메뉴를 정할 수 있는가
- 재료 입력부터 레시피 선택까지 30초 이내
